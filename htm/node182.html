sete 

   'sete' <set> 'n'|'e'|'p'|'l'|'c'|'s'|'b'|'S'|'L'|'se'|'sh' ->
                'max'|'min'|'strict'
This keyword is used to enquire other sets which have entities in common or which have an identical content. Example:

sete blade p min

searches for sets who store at least the same points as "blade" (always the set "all"). The option “max” searches for sets whose entities are completely included in the specified set. The option “strict” searches for identical sets regarding the specified type of entity.
